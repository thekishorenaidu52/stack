## Problems Covered

* Next Greater Element
* Next Greater Element II

* Previous Greater Element
* Previous Smaller Element
* Stock Span Problem
* Largest Rectangle in Histogram
* Maximal Rectangle
* Sliding Window Maximum

## Approach

Most of these problems are solved using the **Monotonic Stack technique**, where a stack maintains elements in increasing or decreasing order to efficiently find the next/previous greater or smaller element.

This technique helps reduce the time complexity from **O(n²) → O(n)**.
